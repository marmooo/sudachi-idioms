# sudachi-idioms
SudachiDict からニ文字熟語や三文字熟語を機械的に生成します。

## Installation
1. install [SudachiDict](https://github.com/WorksApplications/SudachiDict)
2. ```npm install```

## Usage
```
node build.js
```
